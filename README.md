# Proyecto TicTacToe

### Descripción

TicTacToe es una aplicación para Android codificada en lenguaje Kotlin + Jetpack Compose como tarea de clase de la asignatura *Programación multimedia y dispositivos móviles*.

#### Detalles de interés
